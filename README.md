# Nabd Website

واجهة ويب تجريبية لمشروع نبض، مصممة كمنتج مالي حديث بواجهة عربية وإنجليزية ودعم كامل للاتجاهين RTL و LTR.

## التشغيل
افتح ملف `index.html` مباشرة في المتصفح.

## المحتويات
- `index.html`
- `styles.css`
- `script.js`

## الملاحظات
- التصميم Responsive ويعمل على الجوال والتابلت وسطح المكتب.
- لا يحتوي على إيموجيز داخل الواجهة، ويستخدم أيقونات SVG مدمجة.
- الألوان مبنية على هوية قريبة من لون بنك الإنماء مع بيج وذهبي خفيف.
